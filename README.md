A Python tool to encrypt and decrypt files using AES encryption (Fernet).

## Features
- Generate a secure key (`key.key`)  
- Encrypt any file into `.encrypted`  
- Decrypt `.encrypted` files back to original  
- Simple menu-driven interface  

## Requirements
- Python 3.x  
- `cryptography` library
